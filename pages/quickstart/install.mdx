# Install

## Pre-requisites
In order to use Kizzy, you will need the following:

1. A Discord account. If you don't already have one, you can create one for free at [discord.com](https://discord.com).

2. A device running Android 8.1 or higher. Kizzy is compatible with Android 8.1 through 13.
