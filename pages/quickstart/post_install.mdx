# Post Install

<div className="steps-container">

### Open
After installing the Kizzy app, open it on your device.

### Grant Permissions
You will be prompted to grant the app certain permissions. These permissions are necessary for Kizzy to function properly.

| Permission Name | Permission Use Explanation |
|-----------------|----------------------------|
| Usage Access | Allows Kizzy to detect which app you are currently using. |
| Notification Access | Allows Kizzy to detect what music or audio you are listening to. |
| Storage Access | Allows Kizzy to save your custom configs to your storage|


### Login With Discord
Once you have granted the necessary permissions, click on `Login With Discord`
<Video url="/quickstart/login-with-discord.gif"/>

### Change Language
If you wish to change the language, you can do so by clicking on the Language option.
